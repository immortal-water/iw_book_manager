## 项目简介

个人图书管理助手是一款轻量级的 Web 图书管理应用，采用 Python Flask + SQLite + 原生前端技术栈开发，支持本地部署运行。系统提供图书增删改查、模糊搜索、多条件筛选（状态+标签）、批量删除、阅读进度可视化、分页浏览、CSV 数据导出等功能。

## 快速启动

```bash
pip install flask
python main.py
```

浏览器访问 http://127.0.0.1:8000

首次运行时自动建表并初始化示例数据。

## 系统功能

### 核心功能
- 图书增删改查（书名、作者、ISBN、分类、位置、阅读状态）
- 按书名/作者模糊搜索
- 统计看板（总图书数、各分类数量、各状态数量）
- 预设 5 个分类（Java/Python/数据库/前端/其他）

### 扩展功能
- 数据导出 CSV（支持筛选条件，兼容 Excel）
- 批量删除（勾选/全选，一键删除）
- 按阅读状态筛选（未读/在读/已读）
- 阅读进度条可视化（总体进度条 + 单本迷你进度条 + 状态彩色徽章）
- 分页功能（每页 5 条，首页/末页/页码导航）
- 自定义标签系统（多对多关联，标签筛选，自动创建标签，搜索+状态+标签三条件组合筛选）
