## 三、功能详细设计

### F-01 添加图书
- **触发方式：** 点击“+ 添加图书”按钮
- **输入字段：** 书名（必填）、作者（必填）、ISBN（选填）、分类（下拉选择）、位置（选填）、状态（默认“未读”）
- **处理逻辑：** 前端表单验证 → AJAX提交 → 后端参数化插入 SQLite → 返回结果 → 页面刷新
- **关联接口：** `POST /book/add`

### F-02 图书列表展示
- **显示形式：** HTML表格，每行显示 ID、书名、作者、分类名称、位置、状态、操作按钮
- **数据来源：** SQLite联表查询（book LEFT JOIN category）
- **关联接口：** `GET /`

### F-05 模糊搜索
- **搜索范围：** 书名（title）、作者（author）
- **实现方式：** SQL LIKE '%关键词%'，支持任意位置匹配
- **关联接口：** `GET /?search=关键词`

### F-06 统计看板
- **展示位置：** 页面顶部
- **更新方式：** 页面加载时异步请求 `/stats` 接口，动态渲染卡片
- **展示内容：** 总图书数 + 每个分类的图书数量

## 四、功能与接口对应关系

| 功能点 | HTTP方法 | 路由 | 数据库操作 |
|--------|----------|------|------------|
| 图书列表 | GET | `/` | SELECT 联表查询 |
| 添加图书 | POST | `/book/add` | INSERT |
| 获取单本图书 | GET | `/book/get/<id>` | SELECT by id |
| 更新图书 | POST | `/book/update/<id>` | UPDATE |
| 删除图书 | GET | `/book/delete/<id>` | DELETE |
| 统计数据 | GET | `/stats` | SELECT COUNT, GROUP BY |
