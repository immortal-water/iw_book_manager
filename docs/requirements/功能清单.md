## 二、功能详细说明

### F-01 添加图书
- **触发方式：** 点击"+ 添加图书"按钮
- **输入字段：** 书名（必填）、作者（必填）、ISBN（选填）、分类（下拉选择）、位置（选填）、状态（默认"未读"）、标签（选填，逗号分隔）
- **处理逻辑：** 前端表单验证 → AJAX提交 → 后端参数化插入 book 表 → 标签查找或创建并建立关联 → 返回结果 → 页面刷新
- **关联接口：** `POST /book/add`

### F-02 图书列表展示
- **显示形式：** HTML表格，每行显示复选框、ID、书名、作者、分类名称、标签、位置、状态徽章、迷你进度条、操作按钮
- **数据来源：** SQLite联表查询（book LEFT JOIN category）+ 标签关联查询
- **关联接口：** `GET /`

### F-05 模糊搜索
- **搜索范围：** 书名（title）、作者（author）
- **实现方式：** SQL LIKE '%关键词%'，支持与状态筛选、标签筛选组合
- **关联接口：** `GET /?search=关键词`

### F-06 统计看板
- **展示位置：** 页面顶部
- **更新方式：** 页面加载时异步请求 `/stats` 接口，动态渲染卡片
- **展示内容：** 总图书数 + 每个分类的图书数量 + 未读/在读/已读各状态数量

### F-09 数据导出CSV
- **触发方式：** 点击"📥 导出 CSV"按钮
- **导出内容：** ID、书名、作者、ISBN、分类、位置、状态（不含标签和进度）
- **特性：** 支持按当前搜索和状态筛选条件导出；写入UTF-8 BOM头解决Excel中文乱码
- **关联接口：** `GET /export/csv`

### F-10 批量删除
- **触发方式：** 勾选图书复选框后点击"🗑 批量删除"按钮
- **交互逻辑：** 勾选任意图书后按钮自动显示并显示选中数量；支持全选/取消全选；确认后一键删除
- **关联接口：** `POST /book/batch_delete`

### F-11 按状态筛选
- **触发方式：** 状态下拉框选择"未读/在读/已读"后自动刷新页面
- **特性：** 可与搜索关键词组合使用；导出CSV时保留筛选条件
- **关联接口：** `GET /?status=未读`

### F-12 阅读进度条
- **展示形式：** 总体进度条（红<30%/橙30-79%/绿≥80%三色）+ 每本图书迷你进度条（0%/50%/100%）
- **进度规则：** 未读=0%，在读=50%，已读=100%
- **状态徽章：** 未读红色、在读橙色、已读绿色

### F-13 分页功能
- **分页规则：** 每页显示5条图书
- **导航功能：** 首页/上一页/数字页码/下一页/末页；当前页绿色高亮；页码过多时省略号折叠
- **信息展示：** "共 X 条，第 Y/Z 页"
- **特性：** 分页与搜索、状态筛选、标签筛选联动

### F-14 标签系统
- **数据模型：** tag表 + book_tag关联表，图书与标签多对多关系
- **标签显示：** 表格标签列显示蓝色标签，点击可筛选
- **标签筛选：** 标签筛选栏显示所有标签，当前选中标签蓝色高亮，点击"全部"清除
- **标签编辑：** 添加/编辑图书时输入逗号分隔的标签名，自动查找已有标签或创建新标签
- **标签清理：** 删除图书时级联删除book_tag关联记录，tag表中未使用的标签保留
- **组合筛选：** 支持搜索+状态+标签三条件同时筛选

## 三、功能与接口对应关系

| 功能点 | HTTP方法 | 路由 | 数据库操作 |
|--------|----------|------|------------|
| 图书列表 | GET | `/` | SELECT 联表查询（含标签关联） |
| 添加图书 | POST | `/book/add` | INSERT book + INSERT/IGNORE book_tag |
| 获取单本图书 | GET | `/book/get/<id>` | SELECT by id + 标签查询 |
| 更新图书 | POST | `/book/update/<id>` | UPDATE book + DELETE旧标签 + INSERT新标签 |
| 删除单本图书 | GET | `/book/delete/<id>` | DELETE book（级联删除book_tag） |
| 批量删除图书 | POST | `/book/batch_delete` | DELETE book WHERE id IN（级联删除book_tag） |
| 统计数据 | GET | `/stats` | SELECT COUNT, GROUP BY（分类+状态） |
| 导出CSV | GET | `/export/csv` | SELECT 联表查询（支持筛选条件） |
