## 1. 系统主流程

```mermaid
flowchart TD
    Start([用户访问首页]) --> Load[加载图书列表和统计看板]
    Load --> Choose{选择操作}
    
    Choose -->|添加图书| Add[打开添加弹窗]
    Choose -->|搜索图书| Search[输入关键词搜索]
    Choose -->|编辑图书| Edit[点击编辑按钮]
    Choose -->|删除图书| Delete[点击删除按钮]
    
    Add --> FillForm[填写图书信息表单]
    FillForm --> SaveAdd[保存至数据库]
    SaveAdd --> Load
    
    Search --> ShowResult[展示搜索结果]
    ShowResult --> Choose
    
    Edit --> PrefillForm[弹窗预填充数据]
    PrefillForm --> SaveEdit[更新数据库]
    SaveEdit --> Load
    
    Delete --> Confirm{确认删除?}
    Confirm -->|是| DelDB[从数据库删除]
    Confirm -->|否| Load
    DelDB --> Load
```

---

## 2. 添加图书流程

```mermaid
flowchart TD
    Start([用户点击“+ 添加图书”]) --> OpenModal[弹出空白表单]
    OpenModal --> Input[填写书名、作者、ISBN、分类、位置、状态]
    Input --> Check{书名和作者非空?}
    Check -->|否| ShowError[浏览器提示“请填写此字段”]
    ShowError --> Input
    Check -->|是| Submit[点击“保存”]
    Submit --> Ajax[前端通过 AJAX 提交至 /book/add]
    Ajax --> Backend[后端接收数据，参数化 SQL 插入]
    Backend --> Success[返回 JSON: success: true]
    Success --> Reload[页面自动刷新]
    Reload --> End([新书出现在列表中])
```

---

## 3. 编辑图书流程

```mermaid
flowchart TD
    Start([点击某图书的“编辑”按钮]) --> GetBook[前端请求 /book/get/id 获取图书 JSON]
    GetBook --> FillModal[弹窗预填充当前图书信息]
    FillModal --> Modify[用户修改字段]
    Modify --> Submit[点击“保存”]
    Submit --> Ajax[前端 POST 至 /book/update/id]
    Ajax --> Backend[后端执行 UPDATE SQL]
    Backend --> Success[返回 success: true]
    Success --> Reload[页面自动刷新]
    Reload --> End([修改后的信息即刻生效])
```

---

## 4. 删除图书流程

```mermaid
flowchart TD
    Start([点击某图书的“删除”按钮]) --> Confirm{弹出确认对话框}
    Confirm -->|取消| End1([流程终止，图书保留])
    Confirm -->|确定| SendDelete[发送 GET /book/delete/id]
    SendDelete --> Backend[后端执行 DELETE SQL]
    Backend --> Success[返回 success: true]
    Success --> Reload[页面自动刷新]
    Reload --> End2([该图书从列表消失])
```

---

## 5. 搜索图书流程

```mermaid
flowchart TD
    Start([用户输入关键词]) --> Submit[点击“搜索”或回车]
    Submit --> Request[浏览器以 GET 方式请求 /?search=关键词]
    Request --> Backend{关键词是否为空?}
    Backend -->|空| All[查询全部图书]
    Backend -->|非空| Filter[执行 SQL:<br/>WHERE title LIKE '%关键词%'<br/>OR author LIKE '%关键词%']
    All --> Render[渲染模板]
    Filter --> Render
    Render --> Result[返回仅包含匹配图书的页面]
    Result --> End([用户看到筛选后的列表])
```

## 6. 统计看板加载流程（异步）

```mermaid
flowchart TD
    PageLoad[页面加载完成] --> Fetch[前端 Fetch /stats 接口]
    Fetch --> Backend[后端执行 COUNT 与 GROUP BY]
    Backend --> JSON[返回 JSON: total, by_category]
    JSON --> Render[前端 JavaScript 动态生成统计卡片]
    Render --> Display[卡片插入页面顶部 stats 区域]
    Display --> End([用户看到总图书数和各分类数量])
```
