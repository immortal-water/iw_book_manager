## 1. 系统主流程

```mermaid
flowchart TD
    Start([用户访问首页]) --> Load[加载图书列表、统计看板、总体进度条]
    Load --> Choose{选择操作}
    
    Choose -->|添加图书| Add[打开添加弹窗]
    Choose -->|搜索图书| Search[输入关键词搜索]
    Choose -->|状态筛选| Filter[选择状态下拉框]
    Choose -->|标签筛选| TagFilter[点击标签筛选栏]
    Choose -->|编辑图书| Edit[点击编辑按钮]
    Choose -->|删除图书| Delete[点击删除按钮]
    Choose -->|批量删除| Batch[勾选图书后批量删除]
    Choose -->|导出CSV| Export[点击导出按钮]
    Choose -->|翻页| Page[点击分页导航]
    
    Add --> FillForm[填写图书信息含标签]
    FillForm --> SaveAdd[保存至数据库并建立标签关联]
    SaveAdd --> Load
    
    Search --> ShowResult[展示搜索结果]
    ShowResult --> Choose
    
    Filter --> FilterResult[展示状态筛选结果]
    FilterResult --> Choose
    
    TagFilter --> TagResult[展示标签筛选结果]
    TagResult --> Choose
    
    Edit --> PrefillForm[弹窗预填充数据含标签回填]
    PrefillForm --> SaveEdit[更新数据库和标签关联]
    SaveEdit --> Load
    
    Delete --> Confirm{确认删除?}
    Confirm -->|是| DelDB[从数据库删除，级联清理标签]
    Confirm -->|否| Load
    DelDB --> Load
    
    Batch --> SelectBooks[勾选多本图书]
    SelectBooks --> BatchDelete[点击批量删除按钮]
    BatchDelete --> BatchConfirm{确认批量删除?}
    BatchConfirm -->|是| BatchDelDB[批量从数据库删除]
    BatchConfirm -->|否| Load
    BatchDelDB --> Load
    
    Export --> Download[浏览器下载CSV文件]
    Download --> Load
    
    Page --> Load
```

## 2. 添加图书流程（含标签）

```mermaid
flowchart TD
    Start([用户点击添加图书按钮]) --> OpenModal[弹出空白表单]
    OpenModal --> Input[填写书名、作者、ISBN、分类、位置、状态、标签]
    Input --> Check{书名和作者非空?}
    Check -->|否| ShowError[浏览器提示请填写此字段]
    ShowError --> Input
    Check -->|是| Submit[点击保存]
    Submit --> Ajax[前端通过 AJAX 提交至 /book/add]
    Ajax --> InsertBook[后端参数化 SQL 插入 book 表]
    InsertBook --> HasTags{标签字段是否为空?}
    HasTags -->|是| Success[返回 success: true]
    HasTags -->|否| ParseTags[解析逗号分隔的标签名列表]
    ParseTags --> ForEach[遍历每个标签名]
    ForEach --> FindTag{标签是否存在?}
    FindTag -->|存在| GetTagId[获取已有标签ID]
    FindTag -->|不存在| CreateTag[创建新标签并获取ID]
    GetTagId --> InsertRelation[INSERT OR IGNORE book_tag]
    CreateTag --> InsertRelation
    InsertRelation --> MoreTags{还有标签?}
    MoreTags -->|是| ForEach
    MoreTags -->|否| Success
    Success --> Reload[页面自动刷新]
    Reload --> End([新书出现在列表中，含标签])
```

## 3. 编辑图书流程（含标签更新）

```mermaid
flowchart TD
    Start([点击某图书的编辑按钮]) --> GetBook[前端请求 /book/get/id]
    GetBook --> GetTags[获取图书JSON含标签数组]
    GetTags --> FillModal[弹窗预填充当前信息<br/>标签数组转逗号分隔字符串回填]
    FillModal --> Modify[用户修改字段和标签]
    Modify --> Submit[点击保存]
    Submit --> Ajax[前端 POST 至 /book/update/id]
    Ajax --> UpdateBook[后端执行 UPDATE book]
    UpdateBook --> DeleteOldTags[DELETE book_tag WHERE book_id=?]
    DeleteOldTags --> HasTags{标签字段是否为空?}
    HasTags -->|是| Success[返回 success: true]
    HasTags -->|否| ParseTags[解析逗号分隔的标签名列表]
    ParseTags --> ForEach[遍历每个标签名]
    ForEach --> FindTag{标签是否存在?}
    FindTag -->|存在| GetTagId[获取已有标签ID]
    FindTag -->|不存在| CreateTag[创建新标签并获取ID]
    GetTagId --> InsertRelation[INSERT OR IGNORE book_tag]
    CreateTag --> InsertRelation
    InsertRelation --> MoreTags{还有标签?}
    MoreTags -->|是| ForEach
    MoreTags -->|否| Success
    Success --> Reload[页面自动刷新]
    Reload --> End([修改后的信息即刻生效])
```

## 4. 删除图书流程

```mermaid
flowchart TD
    Start([点击某图书的删除按钮]) --> Confirm{弹出确认对话框}
    Confirm -->|取消| End1([流程终止，图书保留])
    Confirm -->|确定| SendDelete[发送 GET /book/delete/id]
    SendDelete --> Backend[后端执行 DELETE book WHERE id=?]
    Backend --> Cascade[外键级联自动删除 book_tag 关联记录]
    Cascade --> Success[返回 success: true]
    Success --> Reload[页面自动刷新]
    Reload --> End2([该图书从列表消失])
```

## 5. 批量删除流程

```mermaid
flowchart TD
    Start([用户开始操作]) --> Check[勾选图书复选框]
    Check --> UpdateBtn[更新批量删除按钮显示和计数]
    UpdateBtn --> More{继续勾选?}
    More -->|是| Check
    More -->|否| Click[点击批量删除按钮]
    Click --> Confirm{弹出确认对话框}
    Confirm -->|取消| End1([流程终止，图书保留])
    Confirm -->|确定| Collect[收集所有选中的图书ID]
    Collect --> Ajax[前端 POST JSON 至 /book/batch_delete]
    Ajax --> Backend[后端执行批量 DELETE SQL]
    Backend --> Cascade[外键级联自动删除所有关联 book_tag 记录]
    Cascade --> Success[返回 success: true 及删除数量]
    Success --> Reload[页面自动刷新]
    Reload --> End2([选中的图书全部从列表消失])
```

## 6. 搜索图书流程

```mermaid
flowchart TD
    Start([用户输入关键词]) --> Submit[点击搜索或回车]
    Submit --> Request[浏览器以 GET 方式请求 /?search=关键词]
    Request --> Backend{关键词是否为空?}
    Backend -->|空| All[查询全部图书]
    Backend -->|非空| Filter[执行 SQL:<br/>WHERE title LIKE 关键词<br/>OR author LIKE 关键词]
    All --> Render[渲染模板]
    Filter --> Render
    Render --> Result[返回仅包含匹配图书的页面]
    Result --> End([用户看到筛选后的列表])
```

## 9. 三条件组合筛选流程

```mermaid
flowchart TD
    Start([用户设置多个筛选条件]) --> SetSearch[输入搜索关键词]
    SetSearch --> SetStatus[选择状态下拉框]
    SetStatus --> SetTag[点击标签筛选]
    SetTag --> Submit[点击搜索或自动提交]
    Submit --> Request[浏览器 GET 请求携带 search、status、tag 参数]
    Request --> Backend[后端动态构建 WHERE 子句:<br/>title/author LIKE + status = + 标签子查询<br/>三条件用 AND 连接]
    Backend --> Query[执行 SELECT DISTINCT 查询]
    Query --> Render[渲染模板]
    Render --> Result[返回同时满足三个条件的图书]
    Result --> End([用户看到精确筛选后的列表])
```

## 10. 导出CSV流程

```mermaid
flowchart TD
    Start([用户点击导出 CSV 按钮]) --> Collect[前端收集当前搜索词和状态筛选值]
    Collect --> BuildURL[构建导出URL含 search 和 status 参数]
    BuildURL --> Request[浏览器请求 /export/csv]
    Request --> Backend[后端动态构建查询条件<br/>复用搜索和状态筛选逻辑]
    Backend --> Query[执行 SELECT 查询<br/>导出全部匹配数据不分页]
    Query --> Generate[生成CSV内容:<br/>写入UTF-8 BOM头<br/>写入表头行<br/>遍历图书写入数据行]
    Generate --> Response[返回 Response 对象<br/>Content-Disposition: attachment]
    Response --> Download[浏览器触发文件下载]
    Download --> End([用户获得 books_export.csv 文件])
```

## 11. 统计看板与进度条加载流程

```mermaid
flowchart TD
    PageLoad[页面加载完成] --> Fetch[前端 Fetch /stats 接口]
    Fetch --> Backend[后端执行统计查询:<br/>COUNT总数<br/>分类 GROUP BY<br/>状态 GROUP BY<br/>计算总体进度百分比]
    Backend --> JSON[返回 JSON:<br/>total, by_category, by_status, overall_progress]
    JSON --> RenderStats[前端动态生成统计卡片:<br/>总图书数 + 各分类数 + 各状态数]
    JSON --> RenderProgress[前端动态更新总体进度条:<br/>设置宽度百分比<br/>根据进度值设置红/橙/绿颜色]
    RenderStats --> Display[渲染到页面]
    RenderProgress --> Display
    Display --> End([用户看到完整的统计信息])
```

## 12. 分页导航流程

```mermaid
flowchart TD
    Start([页面加载或用户操作]) --> Count[后端查询总记录数]
    Count --> Calc[计算总页数和偏移量]
    Calc --> Query[执行 LIMIT 和 OFFSET 查询]
    Query --> Render[渲染当前页图书列表]
    Render --> Nav[渲染分页导航栏<br/>首页/上一页/页码/下一页/末页]
    Nav --> Fold{总页数过多?}
    Fold -->|是| FoldPages[中间页码用省略号折叠]
    Fold -->|否| ShowAll[显示全部页码]
    FoldPages --> Display[展示分页导航]
    ShowAll --> Display
    Display --> UserClick{用户点击导航?}
    UserClick -->|点击页码/导航按钮| Jump[跳转到对应页]
    UserClick -->|无操作| End([浏览当前页])
    Jump --> Count
```
